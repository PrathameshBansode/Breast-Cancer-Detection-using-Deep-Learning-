# 🩺 Breast Cancer Detection using Deep Learning

A deep learning-based web application that classifies breast cancer histopathology images as **Benign** or **Malignant** using a **MobileNet** convolutional neural network. The model incorporates image preprocessing and data augmentation to improve generalization and achieves **94.9% classification accuracy**. The application is deployed using **Flask** for real-time image prediction.

## ✨ Features

- 🔬 Classifies breast cancer histopathology images into **Benign** or **Malignant**
- 🧠 MobileNet-based CNN architecture
- 📈 Achieved **94.9% accuracy**
- 🖼️ Image preprocessing and data augmentation
- 🌐 Flask web application for real-time predictions
- 💻 Simple and user-friendly interface
